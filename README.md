AutoFlow AI:
Autonomous Multi-Agent Workflow System

Problem:
Enterprise workflows require constant monitoring and manual follow-ups.

Solution:
AutoFlow AI uses multiple AI agents to:
- Extract tasks
- Assign priorities
- Monitor workflows
- Detect delays
- Recover automatically

Features:
- Multi-agent architecture
- Autonomous execution
- Delay detection & recovery
- Audit logging

Tech Stack:
- Flask (Backend)
- HTML/CSS/JS (Frontend)

How to Run:
pip install -r requirements.txt  
python app.py
